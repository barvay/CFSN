# CFSN

Codeforces Submission Notifier (CFSN) is a web application that notifies you about the verdict of your latest Codeforces submission.

## Usage

- **Login**: Click the "Login" button to enter your Codeforces handle and start receiving notifications about your submissions.
- **Logout**: Click on "Logout" to stop receiving notifications.
- **Verdicts**: The main section of the application displays the verdicts of your latest submissions in a table format. Each row represents a submission and includes the verdict, problem name, number of tests passed, time consumed, memory consumed, and programming language.
- **Visual Effects**: When a new submission verdict is received, the application will display a visual effect and play a sound based on the verdict.
